# ARTI-308_lab6
In this lab, we applied data analysis and machine learning techniques on the Ecommerce Customers dataset. The process began by loading the dataset into a DataFrame and exploring its structure using functions such as head(), info(), and describe(). Basic data cleaning was performed to ensure data quality, followed by feature selection and preparation for modeling.

A Linear Regression model was then trained using relevant numerical features to predict the yearly amount spent by customers. Finally, the model’s performance was evaluated to assess its accuracy and effectiveness in making predictions.
